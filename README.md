# Quantum AI RAG Assistant - TP2

RAG chatbot on scientific white paper about Quantum AI.
Stack: BAAI/bge-m3 + Qdrant Cloud + Llama 3.3 70B + Streamlit

## Deploy on Streamlit Cloud
1. share.streamlit.io > New app > this repo > app.py
2. Settings > Secrets > add GROQ_API_KEY, QDRANT_URL, QDRANT_API_KEY, COLLECTION_NAME
3. Deploy!
